# Portfolio Analista

Portfolio web personal con foco en analitica, ingenieria de datos y automatizacion.

## Contenido

- Landing page estatica preparada para Vercel.
- Presentacion de experiencia, formacion, skills y proyectos.
- Enlaces directos a repositorios publicos de GitHub.

## Stack

- HTML
- CSS
- JavaScript
- Vercel

## Deploy

Produccion actual:

- `https://portfolio-analista-virid.vercel.app`

## Desarrollo local

Al ser una web estatica, basta con abrir `index.html` o servir la carpeta con cualquier servidor simple.
